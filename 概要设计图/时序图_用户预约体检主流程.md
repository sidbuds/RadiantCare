# 时序图-用户预约体检主流程

```
sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant B as 后端
    participant D as 数据库
    U->>F: 输入账号密码
    F->>B: POST /api/auth/login
    B->>D: 查询用户
    D-->>B: 返回用户信息
    B-->>F: 返回JWT
    F-->>U: 登录成功
    U->>F: 选择套餐和时段
    F->>B: GET /appointments/slots
    B->>D: 查询可用时段
    D-->>B: 返回时段
    B-->>F: 返回数据
    F-->>U: 显示时段
    U->>F: 提交预约
    F->>B: POST /appointments
    B->>D: 创建预约
    D-->>B: 成功
    B-->>F: 返回预约编号
    F-->>U: 预约成功
    U->>F: 创建订单
    F->>B: POST /orders
    B->>D: 创建订单
    D-->>B: 成功
    B-->>F: 返回订单编号
    F-->>U: 订单创建成功
    U->>F: 确认支付
    F->>B: POST /orders/pay
    B->>D: 更新支付状态
    D-->>B: 成功
    B-->>F: 支付成功
    F-->>U: 支付完成
```
