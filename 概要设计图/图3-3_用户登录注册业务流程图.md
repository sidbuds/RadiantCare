# 图3-3 用户登录/注册业务流程图

```
graph TD
    S[开始] --> A{选择操作}
    A -->|登录| B[输入账号密码]
    B --> C{校验输入}
    C -->|不合法| D[提示错误]
    D --> B
    C -->|合法| E{查询数据库}
    E -->|不存在| F[提示用户不存在]
    F --> B
    E -->|密码错| G[提示密码错误]
    G --> B
    E -->|通过| H[返回JWT令牌]
    H --> I[跳转首页]
    A -->|注册| J[输入注册信息]
    J --> K{校验信息}
    K -->|不合法| L[提示错误]
    L --> J
    K -->|合法| M{手机号已注册}
    M -->|是| N[提示已注册]
    N --> J
    M -->|否| O[保存用户信息]
    O --> P[注册成功]
    P --> B
```
