# 熙心健康体检平台 - ER图绘制提示词

## 使用说明

将以下提示词复制到AI绘图工具（如Draw.io、Lucidchart、ProcessOn、AI绘图助手）中生成传统ER图。

---

## 1. 用户体系ER图

```
请绘制一张传统ER图（Chen表示法），包含以下内容：

【实体】（矩形框）
1. 用户（User）
2. 后台账号（StaffAccount）
3. 角色（Role）

【属性】（椭圆形，下划线为主键）
用户：用户ID(主键)、用户编号、姓名、手机号、密码、状态
后台账号：账号ID(主键)、用户名、密码、显示名、绑定用户ID(外键)
角色：角色编码(主键)、角色名称

【关系】（菱形框）
1. 绑定：后台账号 1:1 用户（一个后台账号绑定一个用户）
2. 拥有：后台账号 1:N 角色（一个后台账号可拥有多个角色）

请使用标准ER图符号，布局清晰，连线不交叉。
```

## 传统ER图符号说明

```
┌─────────────────────────────────────────────────────────────┐
│                      符号说明                                │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────┐    矩形 = 实体 (Entity)                        │
│  │  实体   │                                                │
│  └─────────┘                                                │
│                                                             │
│  ╭─────────╮    椭圆 = 属性 (Attribute)                     │
│  │  属性   │    下划线 = 主键 (Primary Key)                  │
│  ╰─────────╯    虚线椭圆 = 外键 (Foreign Key)                │
│                                                             │
│    ◇◇◇◇◇      菱形 = 关系 (Relationship)                   │
│    ◇ 关系 ◇                                                │
│    ◇◇◇◇◇                                                  │
│                                                             │
│  连线标注：                                                  │
│    1:1  一对一                                              │
│    1:N  一对多                                              │
│    N:1  多对一                                              │
│    M:N  多对多                                              │
└─────────────────────────────────────────────────────────────┘
```
