# 熙心健康体检平台

熙心健康体检平台（RadiantCare）用于体检预约、订单、到院导检、结果录入、报告审核发布、在线咨询、历年对比与健康建议的全流程管理。

## 快速启动

### Docker Compose 启动
```powershell
.\docker-up.ps1
```

常用命令：
```powershell
.\docker-up.ps1 -Build
.\docker-up.ps1 -Down
.\docker-up.ps1 -Logs
```
